# Pala Lume

Pala Lume is a responsive salon website built with React and Vite. It allows users to browse services, explore hairstyle inspiration, learn about the salon, and get in touch through a modern, user-friendly interface.

## Features

- Responsive design
- Multi-page navigation
- Reusable React components
- Dynamic service filtering
- Searchable hairstyle gallery
- Contact form
- Responsive navigation menu

## Pages

- Home
- About
- Services
- Gallery
- Contact

## Built With

- React
- React Router DOM
- Vite
- JavaScript
- HTML5
- CSS3
